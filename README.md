#PHP-Regression

## Source
https://codeception.com/quickstart

## Requirement
1. php extension zip, curl

## Installation
1. `composer install`

## Usage
1. Open new terminal
2. `cd <project-root>/webdriver`
3. ./chromedriver --url-base=/wd/hub
4. Open another terminal
5. `cd <project-root>/`
6. `./vendor/bin/codecept run --steps`
